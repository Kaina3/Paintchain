# Phase 06: 回答（推測）フェーズ

## 概要
配られた絵を見て、それが何を表しているかテキストで回答する。

## ゴール
- 前のプレイヤーが描いた絵を表示。
- テキストで回答を入力して提出。
- 制限時間 50 秒、タイムアウトで空文字自動提出。

---

## 成果物
- 回答入力画面
- 絵の表示
- 回答提出 WS イベント

---

## 機能要件

### サーバ側
| イベント | 説明 |
|----------|------|
| `WS: receive_drawing` | 回答すべき絵の URL を受信 |
| `WS: submit_guess` | 回答テキストを提出 |
| `WS: submission_received` | 提出状況通知 |
| `WS: phase_complete` | 全員提出 or タイムアウト |

### フロント側
| 機能 | 説明 |
|------|------|
| 絵の表示 | 前のプレイヤーの描画を表示 |
| 回答フォーム | テキスト入力 + 提出ボタン |
| タイマー | 残り 50 秒カウントダウン |
| 提出状況 | 進捗表示 |
