# Phase 10: モバイル最適化

## 概要
スマートフォンでの描画体験を最適化する。

## ゴール
- タッチ操作で快適に描画できる。
- 画面サイズに応じたレスポンシブ UI。
- モバイル特有の問題（ズーム、スクロール）を解決。

---

## 成果物
- タッチ対応キャンバス
- レスポンシブ UI
- モバイル向け UX 改善

---

## 対応項目

### キャンバス
| 問題 | 対策 |
|------|------|
| 描画中のスクロール | `touch-action: none` で無効化 |
| ピンチズーム | `user-scalable=no` + イベント抑制 |
| 描画遅延 | `requestAnimationFrame` + イベント間引き |
| 指の太さ | タッチポイントを中心に描画 |

### UI レイアウト
| 画面 | 対応 |
|------|------|
| ホーム | 縦長レイアウト、大きめボタン |
| ロビー | プレイヤー一覧をスクロール可能に |
| キャンバス | ツールバーを下部固定、キャンバスを最大化 |
| リプレイ | スワイプで次のスライドへ |
