# クイズモード

お題が自動で出されてその絵を誰か一人が描いて、他の人がその絵がなんなのか当てるモード。制限時間(defo: 120s)あって、その間に親が絵を描いて、他の人が当てる。他の人のguessは画面の上側にニコニコ動画みたいに文字が右から左に流れるようにして、当たってたら正解のマークが流れる。絵を描く人は順番に回る。
点数がつくようになってて、
- 誰かが当てる（上位k人まで）
	- 親に2点
	- 1番目に当てた人に3点
	- 2番目に当てた人に2点
	- 3番目に当てた人に1点
- 誰も当てない
	- 親以外に1点

上記の場合、3人当てたら答えが表示されて他の人もわかるようになる。そして少ししたら次の人へ移行。

# 設定事項

- 絵を描く制限時間(defo: 120s)
- クイズ形式
	- 描いてる最中を見ながら誰かが当てるモード(こちらがdefo)
	- 短い制限時間(defo: 15s)で親が絵を描いてからそれをみんなに見せて当てるモード
		- このモードの場合、15sまでは他の人はキャンバスがロックされてて見えないけど、15s過ぎたら見えるようになって答える、みたいな実装でいいかな。そして全体の時間は30s(描く時間15s, 当てる時間30s)をデフォにして
- お題表示形式
	- お題をいきなり表示していきなり制限時間がスタートするモード
	- お題を見る時間は別にするモード(120sに含まれない)
- 何人まで当てられるか(defo: 3人)
- それぞれの点数
	- 絵が当てられた時と当てられなかった時のそれぞれの人の点数をいじれるようにして
	- なお点数は最大でも一気に3点しか入らないようにして

# 備考
お題は画面では目隠しの状態で表示してクリックしてる時だけ中身が見られるようにして。最初表示するときは数秒の間は見られる状態でいいけど、数秒したら目隠しして。
